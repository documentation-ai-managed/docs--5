## Overview

孙 浩 lets you group docs into projects and sections.

## Steps

1. Click "New Project" on the dashboard
2. Enter your project name (e.g., "Website Docs")
3. Add sections for topics (e.g., "API Guides", "User Flow")
4. Drag and drop docs to rearrange
5. Use the sidebar for quick navigation

<Callout kind="tip">
You can rename, archive, or delete projects anytime from settings.
</Callout>
