## Customization Options

Tailor 孙 浩 to match your brand. The default brand color is `#3B82F6`, but you customize it easily.

<CodeGroup tabs="JSON,YAML">
  ```json
  {
    "theme": {
      "brandColor": "#3B82F6"
    }
  }
  ```
  ```yaml
  theme:
    brandColor: "#3B82F6"
  ```
</CodeGroup>

<Callout kind="tip">
  Update your config file and redeploy. Test the color in preview mode first.
</Callout>

<Expandable title="Advanced Theming" default-open="false">

You extend themes with CSS variables.

````css
:root {
  --brand-color: #3B82F6;
  --accent-color: #1D4ED8;
}
````

</Expandable>
