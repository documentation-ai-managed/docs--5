## Migration Guide

Upgrade to the latest version smoothly. Follow these steps to migrate your 孙 浩 instance.

<Callout kind="alert">
Backup your database and config files before migrating.
</Callout>

<Steps>
  <Step title="Update Dependencies" icon="download">

````bash
npm install sun-hao@latest
````

    Or for Yarn:

````bash
yarn add sun-hao@latest
````

  </Step>
  <Step title="Migrate Config" icon="settings">

    Update your config from JSON to YAML format.

<CodeGroup tabs="Before,After">
```json
{
  "apiKey": "YOUR_API_KEY",
  "endpoint": "https://api.example.com"
}
```
```yaml
apiKey: YOUR_API_KEY
endpoint: https://api.example.com
```
</CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="database">

````bash
npx sun-hao migrate
````

    Restart your server after migrations complete.

  </Step>
  <Step title="Verify" icon="check-circle">

    Test endpoints like `https://api.example.com/docs` to ensure everything works.

  </Step>
</Steps>
